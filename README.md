# byterati.com

Marketing site + App Store support and privacy pages for **Byterati**, a native
Apple app that reads public-domain books aloud.

Static HTML, no build step. Hosted on GitHub Pages at https://byterati.com.

- `index.html` — landing page
- `privacy.html` — Privacy Policy (App Store Connect: Privacy Policy URL)
- `support.html` — Support / FAQ (App Store Connect: Support URL)
- `styles.css` — shared styles
- `CNAME` — custom domain for GitHub Pages
